# Dyslexia-PDF-Reader
PDF Reader for Dyslexia

V0.1

Features:
- Font Changing
  - OpenDyslexic
  - OpenDyslexic Bold
  - OpenDyslexic Italic

TO USE:
- Extract the .zip
- pdfjs-5.4.149-dist -> web -> viewer.html
- There you will be able to open a file through the options tab
- The font change is a dropdown at the top toolbar
